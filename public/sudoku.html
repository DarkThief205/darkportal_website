Quick guide

How Sudoku works

Classic Sudoku

Fill every row, column and 3×3 box with digits 1–9. Each digit can appear only once in each row, column and box.

Sum-Doku

Classic Sudoku rules still apply. Every dashed cage must add up to the small target number shown in its top-left cell.

Difficulty

Easy gives more starting numbers and more help. Hard and Extreme give fewer clues, fewer hints and fewer allowed mistakes.

Controls

Use Pencil for notes, Hint for a safe move, and Preview results to reveal the solution. The configure button returns to setup.
