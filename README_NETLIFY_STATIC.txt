Dark Portal — Netlify-ready static build

Deploy: drag and drop this folder or the ZIP file into Netlify.

This version intentionally contains no package.json, package-lock.json, .env, server.js, db.js, routes, or backend-only docs.
Login/profile/dashboard use browser localStorage for a static preview. Live OAuth, SQLite, Discord guild sync, online Tic-Tac-Toe rooms, and API-backed stats require the original backend.
